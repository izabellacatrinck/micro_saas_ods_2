Comparação com bibliotecas R / R#

Como o Pandas tem como objetivo fornecer grande parte da manipulação e análise de dados funcionalidades para as quais as pessoas usam o R, nesta página foi iniciado para fornecer uma análise mais detalhada da linguagem R e de seus muitos terços Bibliotecas de grupo no que se relacionam aos pandas. Em comparações com R e CRAN Bibliotecas, nos importamos com as seguintes coisas:

  • Funcionalidade / flexibilidade: o que pode ou não pode ser feito com cada ferramenta

  • Desempenho: quão rápidas são as operações. Números/benchmarks rígidos são preferencial

  • Facilidade de uso: Uma ferramenta é mais fácil/difícil de usar (talvez você precise o juiz disso, dado o código comparado lado a lado)

Esta página também está aqui para oferecer um pequeno guia de tradução para usuários desses Pacotes R.

match / %in%#

A common way to select data in R is using which is defined using the function . The operator is used to return a logical vector indicating if there is a match or not:%in%match%in%

s <- 0:4
s %in% c(2,4)

The isin() method is similar to R operator:%in%

In [12]: s = pd.Series(np.arange(5), dtype=np.float32)

In [13]: s.isin([2, 4])
Out[13]: 
0    False
1    False
2     True
3    False
4     True
dtype: bool

The function returns a vector of the positions of matches of its first argument in its second:match

s <- 0:4
match(s, c(2,4))

For more details and examples see the reshaping documentation.

subset#

The query() method is similar to the base R function. In R you might want to get the rows of a where one column’s values are less than another column’s values:subsetdata.frame

df <- data.frame(a=rnorm(10), b=rnorm(10))
subset(df, a <= b)
df[df$a <= df$b,]  # note the comma
