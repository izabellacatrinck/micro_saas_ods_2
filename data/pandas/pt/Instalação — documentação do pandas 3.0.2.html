Instalação#

A equipe de desenvolvimento do Pandas distribui oficialmente o Pandas para instalação Pelos seguintes métodos:

  • Disponível no conda-forge para instalação com o gerenciador de pacotes conda.

  • Disponível no PyPI para instalação com pip.

  • Disponível no Github para instalação a partir da fonte.

Nota

Os pandas podem ser instalados de outras fontes além das listadas acima, Mas eles não são gerenciados pela equipe de desenvolvimento do Pandas.

Suporte para versões em Python#

Veja a política de suporte ao Python.

Instalação de pandas#

Instalação com Conda#

Para usuários que trabalham com o gerenciador de pacotes Conda, Pandas podem ser instalados a partir do canal.conda-forge

conda install -c conda-forge pandas

Para instalar o gerenciador de pacotes Conda no seu sistema, recomenda-se a distribuição do Miniforge.

Além disso, recomenda-se instalar e rodar pandas a partir de um ambiente virtual.

conda create -c conda-forge -n name_of_my_env python pandas
# On Linux or MacOS
source activate name_of_my_env
# On Windows
activate name_of_my_env

Dica

Para usuários que são novos em Python, a maneira mais fácil de instalar Python, pandas e o pacotes que compõem a pilha PyData, como SciPy, NumPy e Matplotlib, são com o Anaconda, uma plataforma multiplataforma (Linux, macOS, Windows) Distribuição em Python para análise de dados e computação científica.

No entanto, o pandas de Anaconda não é oficialmente gerenciado pela equipe de desenvolvimento do pandas.

Instalação com pip#

Para usuários que trabalham com o gerenciador de pacotes PIP, pandas podem ser instalados a partir do PyPI.

pip install pandas

O Pandas também pode ser instalado com conjuntos de dependências opcionais para habilitar certas funcionalidades. Por exemplo, para instalar pandas com dependências opcionais para ler arquivos Excel.

pip install "pandas[excel]"

A lista completa de extras que podem ser instalados pode ser encontrada na seção de dependências.

Além disso, recomenda-se instalar e rodar pandas a partir de um ambiente virtual, por exemplo, usando o venv da biblioteca padrão Python

Instalação a partir da fonte#

Veja o guia de contribuição para instruções completas sobre construção a partir da árvore de origem git. Além disso, veja a criação de um ambiente de desenvolvimento se desejar criar Um ambiente de desenvolvimento Pandas.

Installing the development version of pandas#

Installing the development version is the quickest way to:

  • Try a new feature that will be shipped in the next release (that is, a feature from a pull-request that was recently merged to the main branch).

  • Check whether a bug you encountered has been fixed since the last release.

The development version is usually uploaded daily to the scientific-python-nightly-wheels index from the PyPI registry of anaconda.org. You can install it by running.

pip install --pre --extra-index-url https://pypi.anaconda.org/scientific-python-nightly-wheels/simple pandas

Note

You might be required to uninstall an existing version of pandas to install the development version.

pip uninstall pandas -y

Running the test suite#

If pandas has been installed from source, running will run all of pandas unit tests.pytest pandas

The unit tests can also be run from the pandas module itself with the test() function. The packages required to run the tests can be installed with .pip install "pandas[test]"

Note

Test failures are not necessarily indicative of a broken pandas installation.

Optional dependencies#

pandas has many optional dependencies that are only used for specific methods. For example, pandas.read_hdf() requires the package, while DataFrame.to_markdown() requires the package. If the optional dependency is not installed, pandas will raise an when the method requiring that dependency is called.pytablestabulateImportError

With pip, optional pandas dependencies can be installed or managed in a file (e.g. requirements.txt or pyproject.toml) as optional extras (e.g. ). All optional dependencies can be installed with , and specific sets of dependencies are listed in the sections below.pandas[performance, aws]pandas[all]

Generally, the minimum supported version is ~1 years old from the release date of a major or minor pandas version. Older versions of optional dependencies may still work, but they are not tested or considered supported.

One of the following combinations of libraries is needed to use the top-level read_html() function:

  • BeautifulSoup4 and html5lib

  • BeautifulSoup4 and lxml

  • BeautifulSoup4 and html5lib and lxml

  • Only lxml, although see HTML Table Parsing for reasons as to why you should probably not take this approach.

Warning

  • if you install BeautifulSoup4 you must install either lxml or html5lib or both. read_html() will not work with only BeautifulSoup4 installed.

  • You are highly encouraged to read HTML Table Parsing gotchas. It explains issues surrounding the installation and usage of the above three libraries.
