Dados categóricos#

Esta é uma introdução ao tipo de dado categórico pandas, incluindo uma breve comparação com R's .factor

Categoricals são um tipo de dado pandas correspondente a variáveis categóricas em estatísticas. Uma variável categórica assume uma limitada, e geralmente fixa, número de valores possíveis (; em R). Exemplos são gênero, Classe social, tipo sanguíneo, afiliação nacional, tempo de observação ou classificação via Escala de Likert.categorieslevels

Em contraste com variáveis estatísticas categóricas, dados categóricos podem ter uma ordem (por exemplo, 'concordo fortemente' vs 'concordo' ou 'primeira observação' vs. 'segunda observação'), mas numéricos Operações (adições, divisões, ...) não são possíveis.

Todos os valores dos dados categóricos estão em ou . A ordem é definida por a ordem de , não a ordem lexical dos valores. Internamente, a estrutura de dados consiste em um array e um array inteiro de que apontam para o valor real em O array.categoriesnp.nancategoriescategoriescodescategories

O tipo de dado categórico é útil nos seguintes casos:

  • Uma variável de string composta por apenas alguns valores diferentes. Conversão de tal cadeia Variável para uma variável categórica economiza memória, veja aqui.

  • A ordem lexical de uma variável não é a mesma que a ordem lógica ("um", "dois", "três"). Ao converter para uma categórica e especificar uma ordem nas categorias, ordenação e Min/Max usará a ordem lógica em vez da ordem lexical, veja aqui.

  • Como um sinal para outras bibliotecas Python de que essa coluna deve ser tratada como categórica variável (por exemplo, para usar métodos estatísticos adequados ou tipos de plots).

Veja também a documentação da API sobre categóricos.

Differences to R’s factor#

The following differences to R’s factor functions can be observed:

  • R’s are named .levelscategories

  • R’s are always of type string, while in pandas can be of any dtype.levelscategories

  • It’s not possible to specify labels at creation time. Use afterwards.s.cat.rename_categories(new_labels)

  • In contrast to R’s function, using categorical data as the sole input to create a new categorical series will not remove unused categories but create a new categorical series which is equal to the passed in one!factor

  • R allows for missing values to be included in its (pandas’ ). pandas does not allow categories, but missing values can still be in the .levelscategoriesNaNvalues
