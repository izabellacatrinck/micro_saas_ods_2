In [1]: import pandas as pd
Dados usados neste tutorial:
  • Dados do Titanic

    This tutorial uses the Titanic data set, stored as CSV. The data consists of the following data columns:

    • PassengerId: Id of every passenger.

    • Survived: Indication whether passenger survived. for no and for yes.01

    • Pclass: One out of the 3 ticket classes: Class , Class and Class .123

    • Name: Name of passenger.

    • Sex: Gender of passenger.

    • Age: Age of passenger in years.

    • SibSp: Number of siblings or spouses aboard.

    • Parch: Number of parents or children aboard.

    • Ticket: Ticket number of passenger.

    • Fare: Indicating the fare.

    • Cabin: Cabin number of passenger.

    • Embarked: Port of embarkation.

    To raw data 
    In [2]: titanic = pd.read_csv("data/titanic.csv")

    The string method Series.str.contains() checks for each of the values in the column if the string contains the word and returns for each of the values ( is part of the name) or ( is not part of the name). This output can be used to subselect the data using conditional (boolean) indexing introduced in the subsetting of data tutorial. As there was only one countess on the Titanic, we get one row as a result.NameCountessTrueCountessFalseCountess

Note

More powerful extractions on strings are supported, as the Series.str.contains() and Series.str.extract() methods accept regular expressions, but are out of the scope of this tutorial.

    Whereas replace() is not a string method, it provides a convenient way to use mappings or vocabularies to translate certain values. It requires a to define the mapping .dictionary{from: to}

Warning

There is also a replace() method available to replace a specific set of characters. However, when having a mapping of multiple values, this would become:

titanic["Sex_short"] = titanic["Sex"].str.replace("female", "F")
titanic["Sex_short"] = titanic["Sex_short"].str.replace("male", "M")

This would become cumbersome and easily lead to mistakes. Just think (or try out yourself) what would happen if those two statements are applied in the opposite order…

REMEMBER

  • String methods are available using the accessor.str

  • String methods work element-wise and can be used for conditional indexing.

  • The method is a convenient method to convert values according to a given dictionary.replace
