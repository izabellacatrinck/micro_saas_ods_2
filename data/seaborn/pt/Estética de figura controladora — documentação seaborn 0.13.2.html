Estética de controle da figura#

Desenhar figuras atraentes é importante. Ao fazer números para si mesmo, ao explorar um conjunto de dados, é bom ter gráficos que sejam agradáveis de observar. Visualizações também são centrais para comunicar insights quantitativos ao público, e nesse contexto é ainda mais necessário ter figuras que chamem a atenção e atraiam o espectador.

Matplotlib é altamente personalizável, mas pode ser difícil saber quais cenários ajustar para alcançar um enredo atraente. Seaborn vem com vários temas personalizados e uma interface de alto nível para controlar a aparência das figuras do matplotlib.

import numpy as np
import seaborn as sns
import matplotlib.pyplot as plt

Vamos definir uma função simples para plotar algumas ondas senoidal deslocadas, o que nos ajudará a ver os diferentes parâmetros estilísticos que podemos ajustar.

def sinplot(n=10, flip=1):
    x = np.linspace(0, 14, 100)
    for i in range(1, n + 1):
        plt.plot(x, np.sin(x + i * .5) * (n + 2 - i) * flip)

É assim que a trama fica com os padrões do matplotlib:

sinplot()

Para mudar para os padrões Seaborn, basta chamar a função.

sns.set_theme()
sinplot()

(Note que em versões de seaborn anteriores à 0.8, era chamado de importação. Em versões posteriores, deve ser explicitamente invocado).

Seaborn divide os parâmetros matplotlib em dois grupos independentes. O primeiro grupo define o estilo estético da trama, e o segundo escala vários elementos da figura para que possa ser facilmente incorporada em diferentes contextos.

A interface para manipular esses parâmetros é composta por dois pares de funções. Para controlar o estilo, use as funções e. Para escalar o enredo, use as funções e (e). Em ambos os casos, a primeira função retorna um dicionário de parâmetros e a segunda define os padrões do matplotlib.

Estilos de figura Seaborn#

Existem cinco temas seaborn pré-definidos: , , , , e . Cada um é adequado para diferentes aplicações e preferências pessoais. O tema padrão é . Como mencionado acima, a grade ajuda o gráfico a servir como uma tabela de consulta para informações quantitativas, e o cinza branco sobre a grade ajuda a evitar que a grade concorra com linhas que representam dados. O tema é semelhante, mas é mais adequado para gráficos com elementos de dados pesados:darkgridwhitegriddarkwhiteticksdarkgridwhitegrid

sns.set_style("whitegrid")
data = np.random.normal(size=(20, 6)) + np.arange(6) / 2
sns.boxplot(data=data);

Para muitos gráficos, (especialmente para cenários como palestras, onde você quer usar principalmente figuras para fornecer impressões de padrões nos dados), a grade é menos necessária.

sns.set_style("dark")
sinplot()

sns.set_style("white")
sinplot()

Sometimes you might want to give a little extra structure to the plots, which is where ticks come in handy:

sns.set_style("ticks")
sinplot()

Removing axes spines#

Both the and styles can benefit from removing the top and right axes spines, which are not needed. The seaborn function despine() can be called to remove them:whiteticks

sinplot()
sns.despine()

Some plots benefit from offsetting the spines away from the data, which can also be done when calling despine(). When the ticks don’t cover the whole range of the axis, the parameter will limit the range of the surviving spines.trim

f, ax = plt.subplots()
sns.violinplot(data=data)
sns.despine(offset=10, trim=True);

You can also control which spines are removed with additional arguments to despine():

sns.set_style("whitegrid")
sns.boxplot(data=data, palette="deep")
sns.despine(left=True)

Temporarily setting figure style#

Although it’s easy to switch back and forth, you can also use the axes_style() function in a statement to temporarily set plot parameters. This also allows you to make figures with differently-styled axes:with

f = plt.figure(figsize=(6, 6))
gs = f.add_gridspec(2, 2)

with sns.axes_style("darkgrid"):
    ax = f.add_subplot(gs[0, 0])
    sinplot(6)

with sns.axes_style("white"):
    ax = f.add_subplot(gs[0, 1])
    sinplot(6)

with sns.axes_style("ticks"):
    ax = f.add_subplot(gs[1, 0])
    sinplot(6)

with sns.axes_style("whitegrid"):
    ax = f.add_subplot(gs[1, 1])
    sinplot(6)

f.tight_layout()

Overriding elements of the seaborn styles#

If you want to customize the seaborn styles, you can pass a dictionary of parameters to the argument of axes_style() and set_style(). Note that you can only override the parameters that are part of the style definition through this method. (However, the higher-level set_theme() function takes a dictionary of any matplotlib parameters).rc

If you want to see what parameters are included, you can just call the function with no arguments, which will return the current settings:

sns.axes_style()

You can then set different versions of these parameters:

sns.set_style("darkgrid", {"axes.facecolor": ".9"})
sinplot()

Scaling plot elements#

A separate set of parameters control the scale of plot elements, which should let you use the same code to make plots that are suited for use in settings where larger or smaller plots are appropriate.

First let’s reset the default parameters by calling set_theme():

sns.set_theme()

The four preset contexts, in order of relative size, are , , , and . The style is the default, and was used in the plots above.papernotebooktalkposternotebook

sns.set_context("paper")
sinplot()

sns.set_context("talk")
sinplot()

sns.set_context("poster")
sinplot()

Most of what you now know about the style functions should transfer to the context functions.

You can call set_context() with one of these names to set the parameters, and you can override the parameters by providing a dictionary of parameter values.

You can also independently scale the size of the font elements when changing the context. (This option is also available through the top-level set() function).

sns.set_context("notebook", font_scale=1.5, rc={"lines.linewidth": 2.5})
sinplot()

Similarly, you can temporarily control the scale of figures nested under a statement.with

Both the style and the context can be quickly configured with the set() function. This function also sets the default color palette, but that will be covered in more detail in the next section of the tutorial.
