entorpecido.inf#

Representação em ponto flutuante do infinito (positivo) do IEEE 754.

Retornos

yFlutuador

Uma representação em ponto flutuante do infinito positivo.

Veja também

isinf : Mostra quais elementos são positivo ou negativo infinito

isposinf : Mostra quais elementos são positivos infinitos

isneginf: Mostra quais elementos são negativos infinitos

isnan : Mostra quais elementos não são um número

isfinite : Mostra quais elementos são finitos (não um de Não Número, infinito positivo e infinito negativo)

Notas

O NumPy usa o padrão IEEE para ponto flutuante binário para aritmética (IEEE 754). Isso significa que Não é um Número não é equivalente ao infinito. Além disso, que o infinito positivo não é equivalente ao infinito negativo. Mas infinito é equivalente ao infinito positivo.

Exemplos

>>> import numpy as np
>>> np.inf
inf
>>> np.array([1]) / 0.
array([inf])
entorpecido.Nan#

Representação em ponto flutuante IEEE 754 de Not a Number (NaN).

Retornos

y: Uma representação em ponto flutuante de Não é um Número.

Veja também

isnan: Mostra quais elementos não são um número.

isfinito : Mostra quais elementos são finitos (não um de Não um Número, infinito positivo e infinito negativo)

Notas

O NumPy usa o padrão IEEE para ponto flutuante binário para aritmética (IEEE 754). Isso significa que Não é um Número não é equivalente ao infinito.

Exemplos

>>> import numpy as np
>>> np.nan
nan
>>> np.log(-1)
np.float64(nan)
>>> np.log([-1, 1, 2])
array([       nan, 0.        , 0.69314718])
entorpecido.newaxis#

Um alias conveniente para Nenhum, útil para indexar arrays.

Exemplos

>>> import numpy as np
>>> np.newaxis is None
True
>>> x = np.arange(3)
>>> x
array([0, 1, 2])
>>> x[:, np.newaxis]
array([[0],
[1],
[2]])
>>> x[:, np.newaxis, np.newaxis]
array([[[0]],
[[1]],
[[2]]])
>>> x[:, np.newaxis] * x
array([[0, 0, 0],
    [0, 1, 2],
    [0, 2, 4]])

Produto externo, igual a:outer(x, y)

>>> y = np.arange(3, 6)
>>> x[:, np.newaxis] * y
array([[ 0,  0,  0],
    [ 3,  4,  5],
    [ 6,  8, 10]])

x[np.newaxis, :] é equivalente a e :x[np.newaxis]x[None]

>>> x[np.newaxis, :].shape
(1, 3)
>>> x[np.newaxis].shape
(1, 3)
>>> x[None].shape
(1, 3)
>>> x[:, np.newaxis].shape
(3, 1)
