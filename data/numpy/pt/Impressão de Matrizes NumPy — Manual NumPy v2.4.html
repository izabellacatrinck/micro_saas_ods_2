Impressão de Matrizes NumPy#

Esta página explica como controlar a formatação de arrays impressos do NumPy. Note que essas opções de impressão se aplicam apenas a arrays, não a escalares.

Definindo opções de impressão#

Aplicando configurações globalmente#

Use para mudar as opções de impressão durante toda a sessão de execução. Para inspecionar as configurações atuais de impressão, use:

>>> np.set_printoptions(precision=2)
>>> np.get_printoptions()
{'edgeitems': 3, 'threshold': 1000, 'floatmode': 'maxprec', 'precision': 2, 'suppress': False, 'linewidth': 75, 'nanstr': 'nan', 'infstr': 'inf', 'sign': '-', 'formatter': None, 'legacy': False, 'override_repr': None}

Para restaurar as configurações padrão, use:

>>> np.set_printoptions(edgeitems=3, infstr='inf',
... linewidth=75, nanstr='nan', precision=8,
... suppress=False, threshold=1000, formatter=None)

Aplicando configurações temporariamente#

Use como gerenciador de contexto para substituir temporariamente as configurações de impressão dentro de um escopo específico:

>>> arr = np.array([0.155, 0.184, 0.173])
>>> with np.printoptions(precision=2):
...     print(arr)
[0.15 0.18 0.17]

Todas as palavras-chave que se aplicam a também se aplicam a .

Alterando o número de dígitos de precisão#

O número padrão de dígitos fracionários exibidos é 8. Você pode alterar esse número usando a palavra-chave.precision

>>> arr = np.array([0.1, 0.184, 0.17322])
>>> with np.printoptions(precision=2):
...     print(arr)
[0.1 0.18 0.17]

A opção determina como o cenário é interpretado. Por padrão, exibe valores com o número mínimo de dígitos necessários para representá-los de forma única, usando o mesmo número de dígitos em todos os elementos. Se você quiser mostrar exatamente o mesmo número de dígitos especificado por , use:floatmodeprecisionfloatmode=maxprec_equalprecisionfloatmode=fixed

>>> arr = np.array([0.1, 0.184, 0.173], dtype=np.float32)
>>> with np.printoptions(precision=2, floatmode="fixed"):
...     print(arr)
[0.10 0.18 0.17]

Mudando como a NAN e a infantaria são exibidas#

Por padrão, é exibido como nan e como infant. Você pode sobrescrever essas representações usando as opções e:nanstrinfstr

>>> arr = np.array([np.inf, np.nan, 0])
>>> with np.printoptions(nanstr="NAN", infstr="INF"):
...     print(arr)
[INF NAN  0.]

Controlling scientific notations#

By default, NumPy uses scientific notation when:

  • The absolute value of the smallest number is less than , or1e-4

  • The ratio of the largest to the smallest absolute value is greater than 1e3

    >>> arr = np.array([0.00002, 210000.0, 3.14])
>>> print(arr)
[2.00e-05 2.10e+05 3.14e+00]

To suppress scientific notation and always use fixed-point notation, set :suppress=True

>>> arr = np.array([0.00002, 210000.0, 3.14])
>>> with np.printoptions(suppress=True):
...     print(arr)
[     0.00002 210000.           3.14   ]

Applying custom formatting functions#

You can apply custom formatting functions to specific or all data types using keyword. See numpy.set_printoptions for more details on supported format keys.formatter

For example, to format datetime64 values with a custom function:

>>> arr = np.array([np.datetime64("2025-01-01"), np.datetime64("2024-01-01")])
>>> with np.printoptions(formatter={"datetime":lambda x: f"(Year: {x.item().year}, Month: {x.item().month})"}):
...     print(arr)
[(Year: 2025, Month: 1) (Year: 2024, Month: 1)]
